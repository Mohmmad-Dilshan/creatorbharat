## 💎 Design Standards (Non-Negotiable):
- **Glassmorphism**: 15-20% opacity blurs.
- **Micro-Animations**: Every button click/hover must feel alive.
- **App-like Transitions**: Smooth route fading (Already Active ✅).
- **Zero Placeholders**: Use real data and generated images.
